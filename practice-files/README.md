# Git 실습용 더미 파일 세트

경영자AI 밋업 2차 「지식관리」 교안 — **섹션 3 (Git의 운용)** 실습에 사용하는 가상 회사 파일입니다.

## 사용 방법

1. 이 `practice-files` 폴더를 통째로 복사해 바탕화면 등 편한 곳에 둡니다.
2. 교안 섹션 3의 안내에 따라 이 폴더에서 `git init`부터 시작합니다.
3. 실습을 망쳐도 괜찮습니다 — 폴더를 지우고 다시 복사하면 처음부터 다시 할 수 있습니다.

## 구성

| 파일 | 역할 |
|---|---|
| `견적서_초안.md` | 커밋 실습용 — 내용을 고쳐가며 버전이 쌓이는 것을 봅니다 |
| `회의록_0715.md` | 두 번째 커밋, 브랜치 실습에 사용 |
| `제품사양.md` | 브랜치에서 수정 → 병합(merge) 실습에 사용 |
| `logo.png` | 바이너리 파일 커밋메시지 관리 실습에 사용 |

> 이 파일들의 회사·인물·금액은 모두 가상입니다.
